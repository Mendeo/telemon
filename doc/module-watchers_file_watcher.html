(static) event(input, callbacks)

Следит за изменениями в заданном каталоге или конкретном файле.
Parameters:
Name Type Description
input object Объект с входными параметрами.
Properties
Name Type Attributes Description
path string Путь к отслеживаемому каталогу или файлу.
pre pre <optional>
 Функция предобработки содержимого файла, в котором обнаружены изменения (до вызова функции trigger).
trigger trigger <optional>
 Функция, для принятия решения о том, сообщать об обнаруженном изменении в файле или нет.
post post <optional>
 Функция для постобработки содержимого файла (после вызова функции trigger).
header string <optional>
 Заголовок. Добавляется к финальному сообщению.
timeout number <optional>
 Время (мс), на которое следует прервать отслеживание изменений, после отправки очередного сообщения.
callbacks Array.<callback> Массив функций обратного вызова, куда будет отправляться сообщение об изменениях в директории или файле.
